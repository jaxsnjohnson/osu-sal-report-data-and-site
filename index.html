About OSU Salary Transparency

Democratizing access to public records.

Why this exists

Public records should be truly public—not just available, but accessible. This project is a civic hacking experiment designed to liberate data from static PDFs and make it searchable, analyzable, and transparent for the entire community.

⚠️ Data Disclaimer

This data is parsed automatically from official OSU PDF reports. While we strive for 100% accuracy, the OCR conversion process can occasionally result in errors.

Note: This dataset generally does not include bonuses, per diem, or student employees. Name changes may not be tracked across different reports. Always refer to official documentation for official purposes.

Definitions

  • Classified: Roles represented by SEIU 503 (Sublocal 083).
  • Unclassified: Faculty, professional faculty, and administrators not represented by the union.
